Footer

The footer shows copyright information and the last time the website was updated.

Example
Copyright © 2025 GreenScape. All Rights Reserved.
Code
<footer>
  <div class="container">
    <div class="row">
      <div class="col">
        <p class="footer-body">Copyright © 2025 GreenScape. All Rights Reserved.</p>
      </div>
    </div>
  </div>
</footer>
Usage
  • The footer must stay at the bottom of every page.
  • The footer should include copyright information and the last year the website was updated.
Accessibility Considerations

This primary footer uses dark text against light backgrounds to meet the WCAG color contrast requirement. If using darker backgrounds (surface-brand-primary, surface-brand-secondary), use lighter text colors (text-brand-primary-light, text-inverse).
