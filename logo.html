Logo Variations

Color

The color logo is the primary logo and should be used against light backgrounds.

Grayscale

The grayscale logo should be used when printing in color is not an option.

Color Inversed

The color inversed logo should be used against dark backgrounds to ensure legibility.

Favicon

The favicon logo should be used as a favicon to be displayed on a browser's tab next to a page title.
