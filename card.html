Card

The cards serve as a clickable element that is linked to a project page.

Example
Card Title

Lorem ipsum dolor sit amet consectetur. Duis purus eget interdum in sollicitudin. Lorem ipsum dolor sit amet consectetur. Duis purus eget interdum in sollicitudin.

Button
Code

<div class="card shadow-sm rounded border-0" style="max-width: 400px;">
  <img src="https://via.placeholder.com/400x300" class="card-img-top" alt="Card Image">
  <div class="card-body bg-light">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Lorem ipsum dolor sit amet consectetur...</p>
    <hr>
    <div class="d-flex justify-content-center">
      <a href="#" class="btn btn-warning fw-bold px-4">Button</a>
    </div>
  </div>
</div>
Usage
  • The cards must include a project overview and an associated image.
  • The entire surface of the cards must be clickable.
Accessibility Considerations

The image in the card must include alt text. This card uses dark text on light backgrounds to meet the WCAG color contrast requirements. Use appropriate contrast if changing backgrounds.
