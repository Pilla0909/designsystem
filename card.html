Usage
  • The cards must include a project overview and an associated image.
  • The entire surface of the cards must be clickable.
Accessibility Considerations

The image in the card must include alt text. This primary card uses dark text against light backgrounds to meet the WCAG color contrast requirement. If using any darker backgrounds (surface-brand-primary, surface-brand-secondary), use lighter text (text-brand-primary-light, text-inverse).
