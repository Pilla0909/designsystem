Card

The cards serve as a clickable element that is linked to a project page.

Example

Card Title

Lorem ipsum dolor sit amet consectetur. Duis purus eget interdum in sollicitudin. Lorem ipsum dolor sit amet consectetur. Duis purus eget interdum in sollicitudin.

Code


<div class="card" style="max-width: 375px;">
  <img src="imageplaceholder.png" class="card-img-top" alt="Card image">
  <div class="card-body bg-light">
    <h5 class="card-title text-brand">Card Title</h5>
    <p class="card-text text-muted">
      Lorem ipsum dolor sit amet consectetur. Duis purus eget interdum in sollicitudin.
    </p>
    <hr />
    <div class="d-grid">
      <button class="btn btn-warning fw-bold" type="button">Button</button>
    </div>
  </div>
</div>

Usage

  • The cards must include a project overview and an associated image.
  • The entire surface of the cards must be clickable if used as links.

Accessibility Considerations

The image in the card must include alt text. This primary card uses dark text against light backgrounds to meet the WCAG color contrast requirement. If using any darker backgrounds, use lighter text.
